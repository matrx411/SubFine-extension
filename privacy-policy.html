SubFine – Privacy Policy

Last Updated: February 2026

Summary: SubFine does not collect, store, or share any personal data. All subtitle processing is done locally in your browser.

1. What is SubFine?

SubFine is a free browser extension that helps users manage, display, and translate subtitles on video streaming websites. It works entirely within your browser without sending your personal data to any external servers.

2. Data We Do NOT Collect

SubFine does not collect, store, or transmit:

  • Your name, email, or any personal identifiers
  • Browsing history or visited URLs
  • Video content you watch
  • Subtitle content you upload or display
  • Any behavioral analytics or tracking data

3. How SubFine Works

SubFine operates entirely locally inside your browser. When you upload a subtitle file or use the AI translation feature, all processing happens on your device. The extension only contacts external services in the following limited cases:

  • AI Translation: Subtitle text may be sent to Google Translate API solely for the purpose of translation. No user-identifying data is included in these requests.
  • Remote Notifications: The extension may fetch a JSON configuration file from GitHub to display update notifications. No personal data is sent.
  • QR Code Generation: When viewing a crypto donation address, a QR code image is generated via an external API using only the wallet address string.

4. Permissions Explained

  • activeTab / scripting: Required to detect and interact with video players on the page you are currently viewing.
  • storage: Used to save your personal preferences (e.g., subtitle font size, filter rules) locally in your browser. This data never leaves your device.
  • downloads: Used exclusively to save subtitle files (SRT/VTT format) to your computer when you request it. The extension does not download any video content.
  • webRequest: Used to intercept subtitle file requests made by video players in order to process and display them through SubFine's custom renderer.

5. Third-Party Services

SubFine may interact with the following external services solely to provide its core features:

  • Google Translate API – for optional subtitle translation
  • GitHub (raw.githubusercontent.com) – for optional remote configuration and notifications
  • api.qrserver.com – for optional QR code generation for donation addresses

SubFine is not affiliated with any of these services, and your use of the extension is subject to the respective privacy policies of these providers.

6. Children's Privacy

SubFine does not knowingly collect any information from children under 13 years of age.

7. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date at the top. Continued use of the extension after changes constitutes your acceptance of the updated policy.

8. Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.
